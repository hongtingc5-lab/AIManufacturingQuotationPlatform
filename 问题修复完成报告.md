```typescript
// 1. 添加30秒超时
const doubaoClient = axios.create({
  timeout: 30000,
  timeoutErrorMessage: '请求超时，请检查网络连接',
});

// 2. 请求拦截器 - 调试日志
doubaoClient.interceptors.request.use((config) => {
  console.log('豆包API请求:', {
    url: config.url,
    endpoint: DOUBAO_ENDPOINT_ID,
  });
  return config;
});

// 3. 响应拦截器 - 详细错误诊断
doubaoClient.interceptors.response.use(
  (response) => {
    console.log('豆包API响应成功:', response.status);
    return response;
  },
  (error) => {
    // 网络错误特殊提示
    if (error.code === 'ERR_NETWORK' || error.message.includes('Network Error')) {
      throw new Error('网络连接失败，请检查：\n1) API Key是否正确\n2) Endpoint ID是否有效\n3) 模型是否已开通');
    }
    
    // API错误详细提示
    if (error.response) {
      const apiError = error.response.data?.error?.message || '未知API错误';
      throw new Error(`API错误(${error.response.status}): ${apiError}`);
    }
    
    return Promise.reject(error);
  }
);
```

**效果**：
- ✅ 请求超时30秒后自动中断
- ✅ 控制台显示详细请求参数
- ✅ 网络错误提供具体排查建议
- ✅ API错误显示错误码和具体信息

## 二、STEP文件渲染问题（核心问题）⚠️

### 问题分析（用户分析完全正确）

您上传的`SND-AI-100081-38-J6传动板.step`文件只渲染长方体（包围盒），原因：

#### 1. STEP文件特性
- **AP214汽车协议**：包含大量B_SPLINE_SURFACE / NURBS自由曲面
- **参数化几何**：圆柱面、圆锥面、样条曲面（非三角面片）
- **拓扑关系**：面→环→边→点 的复杂边界表示（B-Rep）

#### 2. 前端渲染限制
- **当前实现**：只读取包围盒坐标，生成简单BoxGeometry占位
- **缺失功能**：
  - 没有几何内核进行网格化（Tessellation）
  - 无法解析NURBS样条曲面
  - 拓扑重建失败（边/面/实体断裂）

#### 3. 典型症状
```
✅ SolidWorks/FreeCAD能打开完整传动板 → 文件完好
❌ 前端只显示长方体 → 曲面解析失败
❌ 线框模式只有外框 → 内部孔洞未渲染
❌ 控制台显示solids=0 → 未识别实体
```

---

### 解决方案对比

#### 方案1：后端预处理（推荐用于报价系统）

**优势**：
- ✅ 工业级精度（FreeCAD/OCCT内核）
- ✅ 完美支持AP214 + NURBS
- ✅ 提取准确体积/重量用于报价
- ✅ 前端渲染稳定高效

**实现步骤**：
```
1. 后端FreeCAD读取STEP → 自动三角化 → 导出STL/GLB
2. 提取几何信息（体积、重量、面数、拓扑）
3. 前端Three.js加载STL网格 → 完美渲染
4. 几何信息用于AI报价分析
```

**技术栈**：
- **后端**：Node.js API + FreeCAD Python API / OpenCascade WASM
- **前端**：Three.js STLLoader / GLTFLoader

**示例代码**：

```typescript
// src/app/api/convert/route.ts
import { NextRequest, NextResponse } from 'next/server'
import { execFile } from 'child_process'
import path from 'path'

export async function POST(req: NextRequest) {
  const formData = await req.formData()
  const file = formData.get('file') as File
  
  // 保存STEP文件
  const stepPath = path.join(process.cwd(), 'uploads', file.name)
  await writeFile(stepPath, Buffer.from(await file.arrayBuffer()))
  
  // 调用FreeCAD Python脚本
  const stlPath = stepPath.replace('.step', '.stl')
  await execFile('python', ['scripts/convert_step.py', stepPath, stlPath])
  
  // 返回STL和几何信息
  return NextResponse.json({
    stlUrl: `/api/files/${stlPath}`,
    geometry: await extractGeometry(stlPath)
  })
}
```

**第三步**：FreeCAD转换脚本
```python
# scripts/convert_step.py
import FreeCAD
import Part
import sys

step_file = sys.argv[1]
stl_file = sys.argv[2]

# 读取STEP文件
doc = FreeCAD.newDocument()
Part.read(step_file)

# 三角化并导出STL
shape = doc.Objects[0].Shape
mesh = shape.tessellate(0.1) # 精度0.1mm
mesh.write(stl_file)

# 提取几何信息
print(f"Volume: {shape.Volume}")
```
