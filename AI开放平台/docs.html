接入文档

使用 OpenAI 或 Anthropic 官方 SDK 接入统一网关。以下域名是演示占位，部署时替换为你的正式 API 域名。

快速开始鉴权方式OpenAI 协议Anthropic 协议模型映射计费规则

快速开始

在控制台创建 API Key，然后将请求地址指向统一网关。

curl https://api.yourdomain.com/v1/chat/completions -H 'Authorization: Bearer sk-nova-your-key'

鉴权方式

所有请求通过 Bearer Token 鉴权。密钥只在创建时完整显示一次，请勿写入前端代码或公开仓库。

Authorization: Bearer sk-nova-your-key

OpenAI 兼容协议

现有 OpenAI SDK 只需替换 baseURL 和密钥。

import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: process.env.NOVA_API_KEY,
  baseURL: 'https://api.yourdomain.com/v1'
});

const result = await client.chat.completions.create({
  model: 'deepseek-v3',
  messages: [{ role: 'user', content: '写一个产品标题' }]
});

Anthropic 兼容协议

支持 Messages API 和流式 SSE，可用于 Claude Code 等兼容客户端。

export ANTHROPIC_BASE_URL=https://api.yourdomain.com
export ANTHROPIC_AUTH_TOKEN=sk-nova-your-key
claude

模型映射

gpt-4.1、claude-sonnet-4、gemini-2.5-pro、deepseek-v3。后台可配置模型别名。

计费规则

每次请求分别统计输入、缓存命中和输出 Token。先扣套餐赠送额度，额度用完后从余额按模型单价扣除；请求失败不收取模型费用。
